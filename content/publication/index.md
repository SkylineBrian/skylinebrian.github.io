## How to add your files
- Put PDFs in: `static/papers/` → link as `/papers/yourfile.pdf`  
- Put BibTeX in: `static/bib/` → link as `/bib/yourfile.bib`  
- Put data in: `static/data/` → link as `/data/yourdataset/`  
- Put code docs (if any) in: `static/code/` **or** link to GitHub repos
